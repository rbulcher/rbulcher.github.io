How to Get Your OpenAI API Key

Step-by-Step Guide

1. Create an OpenAI Account

If you don't already have an OpenAI account, follow these steps:

  1. Go to OpenAI's signup page.
  2. Enter your email address and create a strong password.
  3. Verify your email address by clicking the link sent to your inbox.
  4. Complete any additional verification steps (like phone verification) if prompted.

2. Generate Your API Key

Once you're logged in, follow these steps to create your API key:

  1. Navigate to the API keys section.
  2. Click on the "+ Create new secret key" button.
  3. Optionally, give your key a name for easy identification.
  4. Click "Create secret key" to generate the key.
  5. Copy your new API key immediately and store it securely. You won't be able to see it again!

3. Increase Your Usage Limit

To use the API effectively, you'll need to increase your usage limit:

  1. Go to the billing section of your OpenAI account.
  2. Click on "Set up paid account" or "Add payment method" if you haven't already.
  3. Follow the prompts to add a payment method and set up a pay-as-you-go plan.
  4. Once set up, you'll have a higher usage limit and can start using your API key.

4. Use Your API Key in InboxIQ

Now that you have your API key, you can use it with InboxIQ:

  1. Open the InboxIQ extension in your browser.
  2. Navigate to the settings or options page within the extension.
  3. Look for a field labeled "OpenAI API Key" or similar.
  4. Paste your API key into this field.
  5. Save your settings.

Remember: Your API key is like a password. Never share it publicly or with anyone you don't trust. If you believe your key has been compromised, return to the OpenAI dashboard and revoke it immediately, then create a new one.
